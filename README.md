## What you get

- **Multi-panel grid** (2–36 panels) to review many shots at once  
- **AutoBrowse** — timed hands-free paging across the grid  
- **Slideshow** — 200 transitions, second display / HDMI, folder music  
- **Free rotation** — middle mouse + wheel (0.30° steps), smooth edges  
- **Zoom** down to 5% of native size  
- **Camera RAW** via LibRaw (DNG, CR2, CR3, NEF, ARW, …) + WIC fallback  
- IPTC metadata, ratings, AI Photo Chat, voice commands, and more  

## Install (Windows 10/11 64-bit)

1. Open **[Releases](https://github.com/jacksonkirka/PhotoNavigator-Releases/releases/latest)**  
2. Download **`PhotoNavigator-Setup.exe`**  
3. Run the installer (optional: Install Ollama for local Photo Chat)  

**Free during beta** — no license payment required while PhotoNavigator is in pre-release.
